# Geologix Knowledge Base Index

This document serves as the central directory for the Geologix Knowledge Base.

## 🚀 Quick Start

Open PowerShell in `geologix-ai/` and run:

```powershell
cd "Q:\Dev\Google Avinity\geologix_knowledge_base\geologix-ai"
.\verify-installation.ps1
.\start-geologix.ps1
```

Optional (public access via ngrok):

```powershell
.\start-ngrok.ps1
```
